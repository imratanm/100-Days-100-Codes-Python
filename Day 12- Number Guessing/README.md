# Number Guessing Game
Hello there!
Ever played a number guessing game? This program brings that classic fun to life.

What This Program Demonstrates

 - Functions – Breaking logic into reusable pieces
 - Variable Scope – Understanding local and global variables
 - Recursion – Using recursive calls for repeated actions

How to Play
Guess the number chosen by the computer within a limited number of attempts. Choose your difficulty level and enjoy the challenge!

Have fun!
